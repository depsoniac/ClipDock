## Descripción para el README / landing

Mira Media Studio es una app de escritorio para creadores y editores que necesitan bajar, convertir, mejorar y organizar medios sin brincar entre diez herramientas distintas. Pega un enlace, suelta archivos o manda resultados directo a Premiere: Mira se encarga de la cola, los formatos, los modelos de IA y los componentes externos.

## Tagline
