Mira Media Studio · 0.5.11

Descarga, convierte y mejora medios sin brincar entre apps.

Una mesa de trabajo multimedia para pegar enlaces compatibles, soltar archivos, procesar imágenes/video con IA local, organizar la cola y enviar resultados directo a Adobe Premiere.

Empezar ahora ↓ Ver funciones
Electron + Python FFmpeg / yt-dlp Real-ESRGAN / rembg Puente CEP para Premiere

Todo lo pesado, en una interfaz limpia.

La app separa descargas, laboratorio IA, conversión/recorte, cola y ajustes para que cada tarea tenga su lugar sin saturar la pantalla.

01

Descargas por sesión

Pega enlaces compatibles, analiza, conserva varios elementos y aplica una misma receta con ajuste universal.

02

Sin recodificar por defecto

Cuando la calidad ya existe, descarga directo para ahorrar tiempo y evitar procesos innecesarios.

03

Imagen con IA

Escalado, eliminación de fondo, lienzo, formato de salida, calidad y metadatos en una sola ventana.

04

Video listo para edición

Recorta, convierte a H.264, H.265, ProRes, GIF o audio y manda resultados a Premiere.

Inicio rápido

Arranca en local.

Requisitos sugeridos: Windows 10/11, Node.js, Python compatible y componentes multimedia instalables desde Ajustes.

1
Clona o descarga el repoMantén el proyecto en una carpeta sencilla, sin caracteres raros.
2
Ejecuta setupInstala dependencias iniciales y prepara el motor.
3
Abre MiraUsa npm start o ARRANCAR.bat para entrar a la interfaz.
./setup.ps1
npm start

# Alternativa rápida en Windows
ARRANCAR.bat
Uso responsable

Mira está pensado para contenido propio, material con permiso, recursos internos o fuentes donde la descarga esté permitida. No debe usarse para evadir DRM, pagos, restricciones de acceso o derechos de autor.
